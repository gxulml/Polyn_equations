# Polyn_equations

Ramsey theory and Nullstellensatz

函数（文件）说明：

1、coefmatrix（主程序）

功能：求R(3,3)=6对应方程组的希尔伯特零点定理证书（只是求出系数矩阵而已）。

输入：图的顶点数和多项式最高阶数。

      其中，vn为顶点数；

            d为最高阶数。

输出：系数矩阵。




2、coef_one

功能：求多项式中，指定x的一阶单项式的系数。

输入：多项式、x和x下标，等。

      其中，g为给定的多项式；

            g1为多项式的常数项；

            x为变量数组，k为欲求系数的x的下标，x(k)即为x_k；

            a为变量数；

            d为单项式最高阶数。

输出：x_k的系数。
       


3、coef_high

功能：求多项式中，指定高阶（大于1阶）单项式t的系数。

输入：多项式、单项式t，等。

      其中，g为给定的多项式；

            g1为多项式的常数项；

            x为变量数组，x(i)即为x_i；

            a为变量数；

            pows为t的指数数组。

输出：t的系数。




4、powers

功能：递归函数，用于求所有d阶单项式的指数。

输入：设置元素的起始序号，等，

      其中，n为变量数，

            b为规定的第i到第n个元素的和，

            i为设置元素的起始序号，

            powerline为已经设置好第1到第i-1个元素的数组（最开始为空），本函数将设置i及后边的元素
                             最终会是一个d阶多项式的指数序列，并存入power_matrix中，

输出：无，但会设置公共变量。




5、r33func

功能：生成希尔伯特零点定理恒等式左边的多项式。

输入：图的顶点数和多项式最高阶数，等。

     其中，x为边变量，是一组符号变量x(i) ( i = 1,...,n ) ；

           d为最高阶数；

           vn为顶点数；

输出：生成的多项式。
